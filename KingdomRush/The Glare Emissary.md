# The Glare Emissary

The Glare Emissary is a random event in the game [Legends%20of%20Kingdom%20Rush](Legends of Kingdom Rush) in the map [Extradimensional%20Menace](Extradimensional Menace).
= The Glare Emissary =
A bizarre-looking creature carrying a package in its claws flies over us.
By its outlandish appearance, it looks like it belongs to the cult, and we could certainly use some goods...
Options.
Shoot it down!
Requires: ranged unit
We manage to scare the beast with our projectiles, making it flee and drop the package, which is heading straight towards the ground!
Catch!
Requires: Athletic unit
With a swift dive, we caught the package in time.
Upon opening it, we found some loot and various fragile potions.
