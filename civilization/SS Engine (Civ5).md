# SS Engine (Civ5)

Game Info.
A Spaceship part. Requires [Apollo%20Program%20%28Civ5%29](Apollo Program) and 1 Aluminum.
Strategy.
The SS Engine is part of the Spaceship, which must be constructed to win a science victory. It becomes available late in the game, so you want to start building it right away! See the section on Victory for more details.
