# Samurai (Civ4)

The Samurai is the [Unique%20unit%20%28Civ4%29](unique unit) of the [Japanese%20%28Civ4%29](Japanese) [List%20of%20civilizations%20in%20Civ4](civilization).
Strategy.
Compared to the [Maceman%20%28Civ4%29](Maceman) that it replaces, the Samurai has two more [First%20strike%20%28Civ4%29](first strikes) and starts with the [Drill%20%28Civ4%29](Drill I) [Promotion%20%28Civ4%29](promotion), which Macemen cannot get. However, Samurai cannot be built with [Copper%20%28Civ4%29](Copper).
