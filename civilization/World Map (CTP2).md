# World Map (CTP2)

World Map is a [List%20of%20scenarios%20in%20CTP2](scenario) in "[Call%20to%20Power%20II](Call to Power II)". Much like the [Earth%20%28map%29](True Start Location Earth) [map](map)s in the "[Civilization%20%28series%29](Civilization)" games, it allows each [List%20of%20civilizations%20in%20CTP2](civilization) to spawn in a historically and geographically accurate location.
