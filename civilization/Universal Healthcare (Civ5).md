# Universal Healthcare (Civ5)

Universal Healthcare is a Level 1 [Ideology%20%28Civ5%29%23Freedom](Freedom), [Ideology%20%28Civ5%29%23Order](Order), and [Ideology%20%28Civ5%29%23Autocracy](Autocracy) tenet in "".
Game Info.
+1 {{HappinessIcon5}} Local Happiness from every [List%20of%20wonders%20in%20Civ5%23National%20Wonders](National Wonder).
There are twelve National Wonders in the game (including the [Palace%20%28Civ5%29](Palace), which every civ starts with), thus allowing this tenet to provide up to a +12 bonus to {{Happiness5}}.
