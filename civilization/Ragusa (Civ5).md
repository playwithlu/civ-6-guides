# Ragusa (Civ5)

Game Info.
There are two different versions of Ragusa that appear in the game. It was originally Sicilian, but was later replaced with a Croatian city that stands in present-day Dubrovnik. The [Civilopedia](Civilopedia) entries for both versions of Ragusa are given below; the latter was implemented in the 1.0.1.275 patch, which was released in April 2011.
