# Trajan's Column (CivRev)

Trajan's Column is a [Wonder%20%28CivRev%29](wonder) in "[Civilization%20Revolution](Civilization Revolution)", available for download with the Eternal Wonder &amp; Artifact Pack.
Building Trajan's Column gives all of a player's [Legion%20%28CivRev%29](Legion) units +1 attack. It becomes obsolete with [Invention%20%28CivRev%29](Invention).
