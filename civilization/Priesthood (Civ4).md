# Priesthood (Civ4)

Strategy.
Priesthood allows you to build [Temple%20%28Civ4%29](Temples) in cities (if a religion is present), which increase happiness and culture. It also enables the [Oracle%20%28Civ4%29](Oracle) wonder. 
In the [Omen%20%28Civ4%29](Omen)s scenario, the first to discover Priesthood founds [Protestantism](Protestantism).
