# XCOM Foundation (CivBE)

XCOM Foundation is a [Mods%20%28CivBE%29](mod) for "". It was created by [Steam](Steam) user Alterd-Rushnano and first released on July 7, 2015. It adds Lisa J. Carter's XCOM Foundation to the game as a [Sponsors%20%28CivBE%29](sponsor).
XCOM Foundation was featured in a Mod of the Week article published on Civilization.com. The mod does not work with "", though another version that is compatible with the Codex mod was released on December 24, 2017.
