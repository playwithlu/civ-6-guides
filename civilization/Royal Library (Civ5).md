# Royal Library (Civ5)

Game Info.
[Unique%20building%20%28Civ5%29](Unique building) of the [Assyrian%20%28Civ5%29](Assyrian) [Civilizations%20%28Civ5%29](civilization). Replaces the [Library%20%28Civ5%29](Library).
Strategy.
In addition to the output of the Library it replaces, the Royal Library has a Great Work of Writing slot. Once filled with a [Great%20Work%20%28Civ5%29](Great Work), it provides 10 extra [XP](XP) to [Unit%20%28Civ5%29](units) trained in this [City%20%28Civ5%29](city), so place a Great Work here as soon as possible for the maximum benefit.
