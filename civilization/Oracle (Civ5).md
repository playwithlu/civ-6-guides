# Oracle (Civ5)

Strategy.
The Oracle is a solid wonder that is especially useful for strategies that rely on having many [Social%20policies%20%28Civ5%29](Social Policies). Try to build it if you happen to research [Philosophy%20%28Civ5%29](Philosophy) early, unless you have to delay building your [National%20College%20%28Civ5%29](National College).
Furthermore, it offers the second-earliest [Great%20Scientist%20%28Civ5%29](Great Scientist) point in the game (the [Great%20Library%20%28Civ5%29](Great Library) offers the first), and is valuable to anyone pursuing a [Science%20victory%20%28Civ5%29](scientific victory) as well.
