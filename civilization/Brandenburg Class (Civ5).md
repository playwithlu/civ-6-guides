# Brandenburg Class (Civ5)

The Brandenburg Class is a [Unique%20unit%20%28Civ5%29](unique) naval unit of the [German%20%28Civ5%29](German) [Civilizations%20%28Civ5%29](civilization) from the [Scramble%20for%20Africa%20%28Civ5%29](Scramble for Africa) [List%20of%20scenarios%20in%20Civ5](scenario) in "".
Game Info.
This Ironclad is the strongest ship in this scenario. Only the Germans can build it.
Strategy.
Replaces the Ironclad for the Germans. It has a higher combat strength which gives it a greater ability to influence city states.
