# Religious Orders (Civ5)

Religious Orders is a special tech in the [Conquest%20of%20the%20New%20World%20%28Civ5%29](Conquest of the New World) [Scenario%20%28Civ5%29](scenario).
Game Info.
Allows you to build the [Monastery%20%28Civ5%29](Monastery), a building which allows you to quickly spread a city's culture borders.
