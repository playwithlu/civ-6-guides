# Work Boat (Civ4)

The Work Boat is a non-combat [List%20of%20units%20in%20Civ4](unit) which can build [Tile%20improvement%20%28Civ4%29](tile improvements) on the sea. It cannot enter [Ocean%20%28Civ4%29](ocean) spaces outside one's own [Borders%20%28Civ4%29](cultural borders) until [Astronomy%20%28Civ4%29](Astronomy) has been researched.
