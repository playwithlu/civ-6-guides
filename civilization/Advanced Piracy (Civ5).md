# Advanced Piracy (Civ5)

Advanced Piracy is a special [List%20of%20technologies%20in%20Civ5](technology) from the [Conquest%20of%20the%20New%20World%20%28Civ5%29](Conquest of the New World Deluxe) [Scenario%20%28Civ5%29](scenario).
Game Info.
Allows you to build the [Sea%20Beggar%20%28Civ5%29](Sea Beggar), a stronger pirate ship that can heal outside your [Territory%20%28Civ5%29](territory). This technology also allows you to build the {{Gold5}}-boosting [Building%20%28Civ5%29](building) the [Chocolate%20House%20%28Civ5%29](Chocolate House).
