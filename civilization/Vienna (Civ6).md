# Vienna (Civ6)

Vienna is a militaristic [City-state%20%28Civ6%29](city-state) from the [Jadwiga%27s%20Legacy%20%28Civ6%29](Jadwiga's Legacy) [List%20of%20scenarios%20in%20Civ6](scenario) in "[Civilization%20VI](Civilization VI)".
Strategy.
Vienna's bonuses focus largely on speeding up [Units%20%28Civ6%29](unit) production. If Vienna is razed by [Barbarian%20%28Civ6%29](Barbarians), the scenario ends in defeat for all players.
