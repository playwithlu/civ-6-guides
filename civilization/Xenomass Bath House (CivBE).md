# Xenomass Bath House (CivBE)

The Xenomass Bath House is a building introduced is a building in the "[Rising%20Tide](Rising Tide)" expansion pack. It can only be built by combining [Old%20Earth%20Relics%20%28CivBE%29](Old Earth Relics)
This building provides +2 {{cultureBE}}, units in this city heal after 1 turn.
