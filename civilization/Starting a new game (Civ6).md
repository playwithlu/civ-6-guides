# Starting a new game (Civ6)

When starting a new game in "[Civilization%20VI](Civilization VI)", you can choose from a number of options, including an Advanced Setup.
If you select "Play Now" to begin immediately, it will begin a game on a Small Continents [Map%20%28Civ6%29](map) on Prince [Difficulty%20level%20%28Civ6%29](difficulty) with randomly chosen [Leaders%20%28Civ6%29](leaders) and at Standard [Speed%20%28Civ6%29](speed).
