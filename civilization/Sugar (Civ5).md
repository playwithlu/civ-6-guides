# Sugar (Civ5)

Game Info.
Luxury resource.
Strategy.
 Sugar is usually in warm climates, on [Grassland%20%28Civ5%29](Grassland) or [Flood%20Plains%20%28Civ5%29](Flood Plains) tiles. It is also frequently found in [Marsh%20%28Civ5%29](Marshes), in which case the Marsh must be cleared before a [Plantation%20%28Civ5%29](Plantation) can be built to access the Sugar.
