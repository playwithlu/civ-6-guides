# Monarchy (Civ4)

Strategy.
"Boosts happiness."
Monarchy allows the [Hereditary%20Rule%20%28Civ4%29](Hereditary Rule) [List%20of%20civics%20in%20Civ4](civic), which grants happiness in [City%20%28Civ4%29](cities) when military units are stationed in them. It also enables workers to build the [Winery%20%28Civ4%29](Winery) improvement.
