# Aetherium Factory (Civ5)

Game Info.
Unique building of the [Pulias%20%28Civ5%29](Pulias) civilization in the [Empires%20of%20the%20Smoky%20Skies%20%28Civ5%29](Empires of the Smoky Skies) [List%20of%20scenarios%20in%20Civ5](scenario) of "". Replaces the [Aetherium%20Fabricator%20%28Civ5%29](Aetherium Fabricator).
Strategy.
The Aetherium Factory can only be built by Cutler. Unlike the Aetherium Fabricator, which it replaces, the Aetherium Factory gives a bonus to which increases with the number of resources worked by the city it is built in. This bonus stacks with Cutler's trait.
