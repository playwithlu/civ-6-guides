# Religious Freedom (CivWorld)

 is a [Civic%20%28CivWorld%29](Civic) in "[CivWorld](CivWorld)". It provides a +1 [Happiness%20%28CivWorld%29](Happiness) bonus for all [Citizen%20%28CivWorld%29](citizens) and a 25% penalty to Science output. It can be adopted after discovering [Religion%20%28CivWorld%29](Religion).
