# Caio Duilio Class (Civ5)

The Caio Duilio Class is a [Unique%20unit%20%28Civ5%29](unique) naval unit of the [Italian%20%28Civ5%29](Italian) [Civilizations%20%28Civ5%29](civilization) from the [Scramble%20for%20Africa%20%28Civ5%29](Scramble for Africa) [List%20of%20scenarios%20in%20Civ5](scenario) in "".
Game Info.
An Ironclad that specializes in attacking cities. Only the Italians can build it.
Strategy.
Replaces the Ironclad for the Italians. Receives the coastal raider promotion which increases its strength against cities.
