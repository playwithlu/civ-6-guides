# Worker (Civ4)

The Worker is a non-combat [List%20of%20units%20in%20Civ4](unit) which can build [Tile%20improvement%20%28Civ4%29](tile improvements). [List%20of%20civilizations%20in%20Civ4](Civilizations) led by an [Expansive%20%28Civ4%29](Expansive) [Leaders%20%28Civ4%29](leader) get a bonus of +25% hammers to Worker [Production%20%28Civ4%29](production).
The Worker is replaced by the following [Unique%20unit%20%28Civ4%29](unique unit):
In "BTS/[Next%20War%20%28Civ4%29](Next War)" it is supplemented by the [Clones%20%28Civ4%29](Clone) unit.
Strategy.
Use Workers to improve the land around your cities and build [Roads%20and%20railroads%20%28Civ4%29](roads) that help establish trade routes.
