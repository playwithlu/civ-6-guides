# Obelisk (Civ6)

The Obelisk is a basic [Religion%20%28Civ6%29](religious) [Building%20%28Civ6%29](building) from the [Gifts%20of%20the%20Nile%20%28Civ6%29](Gifts of the Nile) [List%20of%20scenarios%20in%20Civ6](scenario) in "[Civilization%20VI](Civilization VI)". It is purchased in the [Holy%20Site%20%28Civ6%29](Holy Site) [District%20%28Civ6%29](district).
Strategy.
Obelisks are basic religious buildings in Gifts of the Nile. Build them first thing after building Holy Sites to improve cities' generation.
