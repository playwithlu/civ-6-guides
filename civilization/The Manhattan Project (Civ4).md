# The Manhattan Project (Civ4)

Strategy.
The Manhattan Project enables [ICBM%20%28Civ4%29](ICBMs) and [Bomb%20Shelter%20%28Civ4%29](Bomb Shelters) for all players. It can be countered by passing the [Elections%20%28Civ4%29%23No%20Nukes](No Nukes resolution) at the [United%20Nations%20%28Civ4%29](United Nations).
