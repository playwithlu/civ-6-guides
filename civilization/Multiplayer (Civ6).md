# Multiplayer (Civ6)

Multiplayer allows you to play "[Civilization%20VI](Civilization VI)" with or against friends and strangers (and possibly [AI](AI) opponents). The game host will decide every setting, including the [Map%20%28Civ6%29](map), the [Speed%20%28Civ6%29](speed), the starting [Era%20%28Civ6%29](era), and which players, if any, will be on teams.
