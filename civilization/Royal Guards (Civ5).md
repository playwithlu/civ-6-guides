# Royal Guards (Civ5)

Royal Guards is a special tech in the [Conquest%20of%20the%20New%20World%20%28Civ5%29](Conquest of the New World) [Scenario%20%28Civ5%29](scenario).
Game Info.
Allows you to build the [Musketeer%20%28Civ5%29](Musketeer), an enhanced version of the standard [Musketman%20%28Civ5%29](Musketman).
