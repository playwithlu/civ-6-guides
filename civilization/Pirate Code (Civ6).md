# Pirate Code (Civ6)

Pirate Code is a Tier III for the [Brigantine%20%28Civ6%29](Brigantine) and a Tier II for the [Sloop%20%28Civ6%29](Sloop) and [Galleon%20%28Civ6%29](Galleon) from the [Pirates%20%28Civ6%29](Pirates) [List%20of%20scenarios%20in%20Civ6](scenario) in "[Civilization%20VI](Civilization VI)". It requires [Exploding%20Shot%20%28Civ6%29](Exploding Shot) for the Brigantine and [Copper-plated%20Hull%20%28Civ6%29](Copper-plated Hull) for the Sloop and Galleon.
Effects.
+1 additional attack per turn.
