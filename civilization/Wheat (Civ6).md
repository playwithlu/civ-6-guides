# Wheat (Civ6)

 Wheat is a [Resource%20%28Civ6%29%23Bonus](Bonus Resource) in "[Civilization%20VI](Civilization VI)". It is found on flatland [Plains%20%28Civ6%29](Plains).
Strategy.
Improved Wheat provides a +4 bonus with [Feudalism%20%28Civ6%29](Feudalism) and [Replaceable%20Parts%20%28Civ6%29](Replaceable Parts), which can help a [City%20%28Civ6%29](city's) Population grow quickly.
