# Seaxman (Civ5)

The Seaxman is a unique unit from the [Fall%20of%20Rome%20%28Civ5%29](Fall of Rome) [List%20of%20Civilization%20V%20scenarios](scenario) of "". Only the [Frankish%20%28Civ5%29](Frankish) civilization can build it.
Game Info.
Strong, front-line land Unit, which can only be built by the Franks. This Unit has a higher Combat Strength than the Swordsman, which it replaces. It starts with the [Amphibious%20%28Civ5%29](Amphibious promotion).
Strategy.
The Seaxman is the Frankish unique unit. It has a slightly higher Combat Strength than the Swordsman, which it replaces. Additionally it does not require any resources. It starts with the Amphibious promotion, allowing it to attack from the ocean and across rivers without penalty.
