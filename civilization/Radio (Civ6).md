# Radio (Civ6)

Radio is a [Modern%20Era%20%28Civ6%29](Modern Era) [Technology%20%28Civ6%29](technology) in "[Civilization%20VI](Civilization VI)". It can be hurried by building a [National%20Park%20%28Civ6%29](National Park).
Strategy.
The next modern miracle is a way to transmit information...through the air. Literally. If ancient generals knew this was possible, how many of them would be scrambling and doing everything they can to discover how to do it?
Well, modern scientists achieve it, and it changes the world once more. The radio, the first medium which uses "wireless" information transfer, is just the beginning. It inspires the ultimate cultural building, the [Broadcast%20Center%20%28Civ6%29](Broadcast Center) (which can transmit both news and music); it also shows people that not all is work and inspires the ultimate entertainment facility: the [Seaside%20Resort%20%28Civ6%29](Seaside Resort).
