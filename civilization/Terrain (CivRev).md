# Terrain (CivRev)

Terrain refers to the physical features of the [Map%20%28CivRev%29](map) in "[Civilization%20Revolution](Civilization Revolution)". Each [Tile%20%28CivRev%29](tile) has its own terrain type, which determines the yields it provides to the [City%20%28CivRev%29](city) that works it and the combat modifiers it applies to [Ground%20unit%20%28CivRev%29](ground units) that occupy it.
