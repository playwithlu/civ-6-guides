# Luboric Factory (Civ5)

Game Info.
Unique building of the [Orlin%20%28Civ5%29](Orlin) civilization in the [Empires%20of%20the%20Smoky%20Skies%20%28Civ5%29](Empires of the Smoky Skies) [List%20of%20scenarios%20in%20Civ5](scenario) of "". Replaces the [Luboric%20Refinery%20%28Civ5%29](Luboric Refinery).
Strategy.
The Luboric Factory can only be built by Curnow. Unlike the Luboric Refinery, which it replaces, the Luboric Factory gives a bonus to which increases with the number of resources worked by the city it is built in. This bonus stacks with Curnow's trait.
