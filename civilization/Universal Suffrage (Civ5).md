# Universal Suffrage (Civ5)

Universal Suffrage is a [Social%20policies%20%28Civ5%29](social policy) in "[Civilization%20V](Civilization V)". It is part of the [Freedom%20%28Civ5%29](Freedom) tree. In "[Brave%20New%20World](Brave New World)" it is a Level 2 [Ideology%20%28Civ5%29%23Freedom](Freedom) tenet.
Game Info.
Normally, Specialists produce the same +1 per one as does any other . This policy halves that. This is equivalent to increasing each 's output by +0.5 .
