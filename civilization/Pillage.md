# Pillage

Pillage is a [unit](unit) command in the "[Civilization%20%28series%29](Civilization)" games that destroys an [Tile%20improvement](improvement) on a [tile](tile) owned by an enemy. Pillaging costs [movement](movement) and provides the unit's owner with [Gold%20%28currency%29](gold). In some games, it also [Healing](heals) the unit.
