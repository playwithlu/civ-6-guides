# Quinquereme (Civ5)

Game Info.
[Unique%20unit%20%28Civ5%29](Unique) naval melee unit of the [Carthaginian%20%28Civ5%29](Carthaginian) [Civilizations%20%28Civ5%29](civilization). Replaces the [Trireme%20%28Civ5%29](Trireme).
Strategy.
The Quinquereme is basically an upgraded Trireme. It has greater combat strength, but otherwise works the same way. It is the only unique [Ancient%20Era%20%28Civ5%29](Ancient-era) ship besides the [Dromon%20%28Civ5%29](Dromon).
